## What To Build

Configure MCP servers in project `opencode.json` for agent tooling:

1. **Laravel Boost** — schema, routes, errors, logs, DB queries
   - `php artisan boost:install` (verify Laravel 11 compat first)
   - Config: `php artisan boost:mcp`
2. **PostgreSQL MCP** — schema introspection, query running
   - `@modelcontextprotocol/server-postgres` (official reference)
   - Connection string from Docker Compose postgres service
3. **Tailwind CSS MCP** — utility lookup, CSS→Tailwind conversion
   - `tailwindcss-mcp-server` (npm)
4. **Docker MCP** — container management, compose, logs
   - `docker-mcp` (uvx)

If Laravel Boost doesn't support Laravel 11, document the gap and configure alternatives.
