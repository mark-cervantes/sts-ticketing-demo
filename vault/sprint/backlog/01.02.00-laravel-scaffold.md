## What To Build

Laravel 13 + Sail base is already scaffolded. This task adds the frontend stack and queueing:

1. **Breeze with Inertia + Vue + TypeScript:**
   - `sail composer require laravel/breeze --dev`
   - `sail artisan breeze:install vue --typescript`
   - `sail npm install`

2. **shadcn-vue:**
   - `sail npx shadcn-vue@latest init`
   - Add starter components: `button dialog sheet input select badge skeleton sonner`

3. **Drag-drop library:** `sail npm install vue-draggable-plus`

4. **Tailwind config:**
   - CSS custom properties for theming (single-file primary color change)
   - Dark mode strategy: `class` selector
   - Primary color token: `--color-primary` → mapped via `@theme inline`

5. **Horizon:**
   - `sail composer require laravel/horizon`
   - `sail artisan horizon:install`

6. **Verify the dev loop:**
   - `sail up -d` brings up all services
   - `sail npm run dev` starts Vite with HMR
   - Login page from Breeze renders at `http://localhost`

## Tests Required
- Breeze default tests pass: `sail artisan test`
