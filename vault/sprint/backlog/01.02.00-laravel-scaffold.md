## What To Build

Initialize the Laravel 11 project with Inertia + Vue 3 + TypeScript:

1. `composer create-project laravel/laravel .` (in project root, alongside existing vault/)
2. `composer require laravel/breeze --dev`
3. `php artisan breeze:install vue --typescript`
4. `npm install`
5. Install shadcn-vue: `npx shadcn-vue@latest init`
6. Install vue-draggable-plus: `npm install vue-draggable-plus`
7. Configure Tailwind with CSS custom properties for single-source theming
8. Set up dark mode in Tailwind config
9. Configure PostgreSQL connection in `.env`
10. Configure Redis connection in `.env`
11. Add Horizon: `composer require laravel/horizon && php artisan horizon:install`
12. Create `docker-compose.yml` with: app, postgres, redis services (basic, for local dev)
13. Verify: `php artisan serve` + `npm run dev` boots without errors
