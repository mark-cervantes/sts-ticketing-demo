## What To Build

Category CRUD for the dynamic DB-backed category system (ADR-006).

### Endpoints
| Method | Route            | Action                  |
| ------ | ---------------- | ----------------------- |
| GET    | /categories      | List all categories     |
| POST   | /categories      | Create new category     |
| DELETE | /categories/{id} | Delete (if no issues)   |

### Controller
**CategoryController** (thin):
- `index()` — return all categories, ordered by name
- `store(StoreCategoryRequest $request)` — create with auto-slug
- `destroy(Category $category)` — delete if no issues reference it

### Form Request
**StoreCategoryRequest**:
- `name`: required, string, trimmed, unique (case-insensitive via Rule::unique with raw expression)

### Business Rules
- Slug auto-generated from name (model boot `creating` event)
- Duplicate name (case-insensitive) → 422
- Delete: if category has issues → 409 with count ("Cannot delete: 5 issues use this category")
- Delete: if no issues → 204

### Response
- List: array of { id, name, slug }
- Create: 201 with { id, name, slug }
- Delete: 204
