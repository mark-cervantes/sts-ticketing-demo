## What To Build

The complete async AI summary generation subsystem (ADR-002, SPEC §7, SRS §7).

### Architecture
```
app/
├── Contracts/SummaryGeneratorInterface.php
├── Services/Summary/
│   ├── SummaryManager.php           ← extends Illuminate\Support\Manager
│   ├── Drivers/
│   │   ├── LlmDriver.php           ← OpenAI-compatible HTTP client
│   │   └── RulesDriver.php         ← Deterministic fallback
│   └── SummaryResult.php            ← Immutable value object
├── Facades/Summary.php
├── Jobs/GenerateSummaryJob.php
├── Events/SummaryCompleted.php
├── Exceptions/SummaryGenerationException.php
└── Providers/SummaryServiceProvider.php
```

### SummaryGeneratorInterface
```php
interface SummaryGeneratorInterface
{
    /** @throws SummaryGenerationException */
    public function generate(Issue $issue): SummaryResult;
}
```

### SummaryResult (Value Object)
```php
final readonly class SummaryResult
{
    public function __construct(
        public string $summary,
        public string $suggestedNextAction,
        public string $driver,
    ) {}
}
```

### SummaryManager
- Extends `Illuminate\Support\Manager`
- `getDefaultDriver()` → `config('summary.default')`
- `createLlmDriver()` → LlmDriver with HTTP client + config
- `createRulesDriver()` → RulesDriver
- Auto-fallback: if `llm` selected but no API key → return rules driver

### LlmDriver
- Laravel HTTP client (injectable, mockable)
- POST `{base_url}/chat/completions`
- Temperature: 0.3, response_format: json_object
- Timeout: configurable (default 30s)
- Throws SummaryGenerationException on any failure

### RulesDriver
- Category + priority matrix → domain-specific summaries
- Description analysis: extract lead sentence
- Action suggestions: concrete, category-aware
- MUST produce genuinely useful output

### GenerateSummaryJob
- tries: 3, backoff: [10, 30, 90]
- Sets summary_status = processing on start
- Calls Summary::generate($issue)
- On success: updates summary + next_action, status = ready
- On final retry failure: fallback to rules driver
- On permanent failure: status = failed
- Fires SummaryCompleted event

### Config
```php
// config/summary.php
return [
    'default' => env('SUMMARY_DRIVER', 'rules'),
    'drivers' => [
        'llm' => [
            'base_url' => env('LLM_BASE_URL'),
            'api_key' => env('LLM_API_KEY'),
            'model' => env('LLM_MODEL'),
            'timeout' => env('LLM_TIMEOUT', 30),
        ],
        'rules' => [],
    ],
];
```

### Prompt Template
Create `config/prompts/summary.php` with system + user messages.

### Service Provider
