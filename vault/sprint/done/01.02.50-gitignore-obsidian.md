## What To Build

Stop tracking Obsidian editor-state files. They cause noise in every PR and have no value in version control.

1. Add to `.gitignore`:
   ```
   .obsidian/
   ```
   (root-level — covers both `./.obsidian/` and `./vault/.obsidian/`)

2. Untrack existing tracked instances:
   ```bash
   git rm -r --cached .obsidian/ vault/.obsidian/ 2>/dev/null || true
   ```

3. Commit: `chore(git): ignore .obsidian editor state`

## Done When

- [x] `.obsidian/` present in `.gitignore`
- [x] `git ls-files | grep '.obsidian/'` returns empty
- [x] Working tree clean after commit
- [x] Merged to dev
